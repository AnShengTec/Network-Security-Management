
# 2017 宁波市安全网络安全管理员培训会议纪要


   自18大后习近平担任中央网络安全和信息化领导小组组长后，为认真贯彻《网络安全法》，推动网络安全
各项制度、责任和技术措施的落实，进一步强化全市网络与信息安全人才队伍建设，结合2017年网络安全宣传周活动，
以此为背景市经信委特主办了本次培训班。
